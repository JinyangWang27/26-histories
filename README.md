# 二十六史

Markdown格式的二十六史，获历朝代纳为正统的史书，又称“正史”。二十六史在二十四史的基础上增加《新元史》和《清史稿》。二十四史上起传说的黄帝，止于明朝崇祯十七年（1644年）。

适用于
- RAG快速构建embedding
- Obsidian

更新至金史
